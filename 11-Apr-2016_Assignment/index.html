"Sigma"

Sigma

Implement function sigma(num)* that given a number, returns the sum of all positive integers up to number (inclusive). Examples: sigma(3) = 6 (or 1 + 2 + 3); sigma(5) = 15 (or 1 + 2 + 3 + 4 + 5).


          function sigma(num){
            if (num === 1){
              return 1;
            } else {
              return num + sigma(num - 1);
            }
          }

"Factorial"

Factorial

Write a function factorial(num) that, given a number, returns the product (multiplication) of all positive integers from 1 up to number (inclusive). For example, calling factorial(3) shoudl return 6 (or 1 x 2 x 3); factorial(5) = 120 (or 1 x 2 x 3 x 4 x 5).


          function factorial(num){
            if (num === 1){
              return 1;
            } else {
              return num * factorial(num - 1);
            }
          }

StarArt - 1

Write a function drawLeftStars(num) that accepts a number and prints that many asterisks.


          function drawLeftStars(num){

              stars += '*';

          }

"Right Stars"

StarArt-2

Write a function drawRightStars(num) that prints 75 characters total. The stars should build from the right side. The last ‘num' characters should be asterisks; the rest of the 75 should be spaces.


          function drawRightStars(num){
            var stars = '';
            for (i = 1; i <= 75-num; i++){
              stars += '-';
            }

            for (var i = stars.length; i <= 75; i++){
              stars += '*';
            }
            console.log(stars);
          }

Write a function drawCenteredStars(num) that prints 75 characters total. The stars should be centered in the 75. The middle num characters should be asterisks; the rest spaces.
